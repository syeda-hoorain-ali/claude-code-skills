# OpenAI Agents SDK + MCP Server

---

### Sub agents

- **MCP Server Expert Agent:** Expert agent for implementing Model Context Protocol (MCP) server integrations with OpenAI Agents SDK, capable of connecting any type of MCP server using the appropriate transport method

---

### Skills

1. **Hosted MCP Server:** Creates and configures hosted MCP server connections for OpenAI Agents SDK
2. **Streamable HTTP MCP Server:** Creates and configures Streamable HTTP MCP server connections for OpenAI Agents SDK
3. **MCP Server SSE:** Creates and configures HTTP with Server-Sent Events MCP server connections for OpenAI Agents SDK
4. **MCP Server stdio:** Creates and configures stdio MCP server connections for OpenAI Agents SDK
5. **MCP Server Troubleshooting:** Diagnoses and resolves common MCP server connection issues with OpenAI Agents SDK
6. **MCP Server Discovery:** Discovers and finds connection details for MCP servers through research and documentation
